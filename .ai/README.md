# AI Context Folder

This hidden folder stores persistent context for future agent sessions.

## Files

- `HANDOFF.md`: rolling session summaries, status, and next steps.
- `DECISIONS.md`: durable engineering decisions and rationale.
- `SESSION_TEMPLATE.md`: copy/paste template for new handoff entries.

## Workflow

1. Start by reading `HANDOFF.md` and `DECISIONS.md`.
2. During work, capture only high-value changes and gotchas.
3. End each session with a concise handoff update.
